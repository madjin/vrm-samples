陰影ベーステクスチャセット 説明書/Shading base Texture Set Description

この度は当サークル「蒼空の下で」のショップ「蒼空の下の市場」にて、「陰影ベーステクスチャセット」の
ダウンロードをしていただき、誠にありがとうございます。

この説明書をお読みいただき、皆様の VRoidライフがより良きものと慣れれば幸いです。

このアイテムは、blender上でVRoid Studioから出力したVRMファイルを元に、パーツごとに
AOベイクを行った後加筆を行って作成しています。

使い方/How to use-------------------------------------
本素材はVRoid Studio v0.10.0以降での使用を前提としてデザインをしています。
（v0.8.0より前のバージョンでの使用は、テクスチャ関連のバグで質感が潰れますので推奨いたしません。）

• PNG素材の使い方
それぞれのパーツのレイヤーにインポートをお願いします。
本素材は、衣装テクスチャ作成時の下地としての使用を前提としています。
インポート後、必要に応じて陰影の加筆や模様を入れるなどの加工をすることを推奨します。

This file is designed to be used in VRoid Studio v0.10.0 or later.
(Use of VRoid Studio v0.8.0 or earlier is not recommended as a texture-related bug may cause blurring of textures.

- How to use PNG material
Please import into each part's layer.
This material is intended to be used as a base for creating costume textures.
After importing, we recommend you to add shade or pattern if necessary.
